Location

Validate against: http://json-schema.org/schema#

ID: https://digitallinguistics.blob.core.windows.net/schemas/location-1.0.0.json#

A location, optionally with accompanying geographic coordinate data. Each location should also be a valid GeoJSON geometry object.

All of these schemas must be matched (allOf)

    An instance of the schema at https://raw.githubusercontent.com/dwhieb/geojson-schema/master/geojson-schema.json#/definitions/geometry
