Contributor

Validate against: http://json-schema.org/schema#

ID: https://digitallinguistics.blob.core.windows.net/schemas/contributor-1.0.0.json#

Type(s): object

A contributor is any person who contributed to the creation or analysis of this text, including speakers, linguists, transcribers, recorders, etc.

Required

  • name
  • role

Additional Properties

true

Properties

  • Abbreviation: "abbreviation"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/abbreviation#

    An abbreviation for this person.

  • Name: "name"

    An instance of the schema at http://digitallinguistics.blob.core.windows.net/schemas/multiLangString#

    The name of this person. If the person is using a pseudonym in your data, you should use that pseudonym in this field.

  • Role: "role"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/multiLangString#

    The role that this person played in this text, e.g. speaker or recorder.

  • Person: "person"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/person#

    A DLx person object containing more detailed information about this person.

Default Value

{}
