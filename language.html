Language

Validate against: http://json-schema.org/schema#

ID: https://digitallinguistics.blob.core.windows.net/schemas/language-1.0.2.json#

Type(s): object

The DLx framework uses the term language to refer broadly to any language variety - whether conceptualized as a language, dialect, or idiolect - at a particular point in time.

Required

  • name

Additional Properties

true

Properties

  • Abbreviation: "abbreviation"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/abbreviation#

    A short, human-readable abbreviation for this language.

  • Language Codes: "codes"

    Type(s): object

    An object containing any ISO 639-3 and Glottolog language codes for this language.

    Required

    • iso
    • glottolog

    Additional Properties

    Properties

    • ISO 639-3 Codes: "iso"

      Type(s): array

      A list of ISO 639-3 codes that are applicable to this language variety.

      Unique items: true

      Items

      ISO 639-3 Code: "iso"

      Type(s):

      Regular expression pattern: [a-z]{3}

    • Glottolog Codes: "glottolog"

      Type(s): array

      A list of Glottolog codes that are applicable to this language variety.

      Unique items: true

      Items

      Glottolog Code: "glottolog"

      Type(s):

      Regular expression pattern: [a-z]{4}[0-9]{4}

  • Locations: "locations"

    Type(s): array

    Geographic locations where this language variety is spoken, optionally with geographic coordinate data.

    Unique items: true

    Items

    Location: "locations"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/location#

  • Language Name: "name"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/multiLangString#

    The name of this language, optionally in multiple orthographies.

  • Phonemes: "phomemes"

    Type(s): array

    The phonological inventory for this language, i.e. a collection of phonemes.

    Items

    Phoneme: "phomemes"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/phoneme#

  • Orthographies: "orthographies"

    Type(s): array

    A collection of orthographies used to represent this language.

    Unique items: true

    Items

    Orthography: "orthographies"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/orthography#

  • URL: "url"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/url#

Default Value

{
  "abbreviation": "",
  "codes": {
    "iso": [],
    "glottolog": []
  },
  "name": "",
  "phonemes": [],
  "orthographies": []
}
