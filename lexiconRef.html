Lexicon Reference

Validate against: http://json-schema.org/schema#

ID: https://digitallinguistics.blob.core.windows.net/schemas/lexiconRef-1.0.0.json#

Type(s): object

A reference to an item in a lexicon.

Additional Properties

Properties

  • Referenced Lexicon: "lexicon"

    Type(s): string

    The name of the lexicon being referenced, or the lexicon that contains the item being referenced.

  • Referenced Lexicon URL: "lexiconUrl"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/url#

  • Referenced Lexeme Key: "lexeme"

    Type(s): string

    The key of the lexeme being referenced.

  • Referenced Lexeme URL: "lexemeUrl"

    An instance of the schema at https://digitallinguistics.blob.core.windows.net/schemas/url#

  • Referenced Sense: "sense"

    Type(s): integer

    The number of the sense being referenced. The sense number is its index in the senses array on the lexeme.

  • Referenced Form: "form"

    Type(s): integer

    The number of the form being referenced. The form number is its index in the forms array on the lexeme.
