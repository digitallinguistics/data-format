Word

Validate against: http://json-schema.org/schema#

Schema ID: https://cdn.digitallinguistics.org/schemas/word-1.0.0.json#

Description

In the DLx framework, a word refers to a specific token in context, rather than an abstract type. Each word maps to one or more morphemes.

Type: object

Required Properties

  • transcription
  • translation

Properties

  • Word Components: "components"

    Description

    A collection of the components (morphemes, lexicon items) in this word. Components may be repeated if the same morpheme appears twice in the word. Note that each component here is a reference to a lexicon item in a lexicon associated with this text. At a minimum, the reference must contain a unique lemma form, lexeme form, or lexeme key.

    Type: array

    Unique items: false

    Items

    Lexicon Item

    Must be an instance of the schema at https://cdn.digitallinguistics.org/schemas/lexiconRef-latest.json#

  • Gloss: "gloss"

    Description

    A Leipzig-style gloss for this word. This does not always need to be supplied if it can be generated automatically from the glosses at the morpheme level.

    Type: string

  • Notes: "notes"

    Description

    A collection of notes about this token.

    Type: array

    Unique items: true

    Items

    Note

    Must be an instance of the schema at https://cdn.digitallinguistics.org/schemas/note-latest.json#

  • Tags: "tags"

    Must be an instance of the schema at https://cdn.digitallinguistics.org/schemas/tags-latest.json#

  • Transcription: "transcription"

    Must be an instance of the schema at https://cdn.digitallinguistics.org/schemas/multiLangString-latest.json#

  • Translation: "translation"

    Must be an instance of the schema at https://cdn.digitallinguistics.org/schemas/multiLangString-latest.json#

  • URL: "url"

    Must be an instance of the schema at https://cdn.digitallinguistics.org/schemas/url-latest.json#

Additional properties: true
