Multi-Language Text / String

Validate against: http://json-schema.org/schema#

Schema ID: http://cdn.digitallinguistics.io/schemas/MultiLangString-2.0.0.json

Description

A piece of text in multiple languages. This is typically used for analyses rather than transcriptions of data (for that, see the Transcription schema). For example, a translation might be given in both English and Spanish; or the part of speech for a lexeme might be given as both English 'noun' and Spanish 'nombre'. Each key must be the abbreviation for a language (see the Abbreviation schema), and the value for that key is the string of text in that language.

Type: object

Additional properties: true

Custom Properties (patternProperties)

  • Text

    Property name must match the regular expression: ^[(a-z)|(A-Z)|(0-9)]+$

    Description

    The string of text in the specified language.

    Type: string
