Note

Validate against: http://json-schema.org/schema#

Schema ID: https://cdn.digitallinguistics.org/schemas/note-1.0.0.json#

Description

A note about this resource. Notes may also include type and language attributes.

Type: object

Required Properties

  • note

Properties

  • Note Type: "type"

    Description

    The type of note.

    Type: string

    Allowed Values (enum)

    • "note-to-self"
    • "general"
    • "anthropological"
    • "discourse"
    • "encyclopedic"
    • "grammar"
    • "phonology"
    • "semantics"
    • "sociocultural"

  • Note Text: "note"

    Description

    The text of the note itself.

    Type: string

  • Note Language: "lang"

    Description

    A language code or language name of the language that this note is written in.

    Type: string

  • Source (Person): "source"

    Must be an instance of the schema at https://cdn.digitallinguistics.org/schemas/person-latest.json#
