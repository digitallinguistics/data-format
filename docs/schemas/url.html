URL

Validate against: http://json-schema.org/schema#

Schema ID: http://cdn.digitallinguistics.io/schemas/url-1.0.2.json

Description

A URL where this resource can be retrieved. The resource does not have to be publicly available at this URL; it may require permissions to access. Best practice is to make the resource available at this URL in JSON format.

Type: string

Format: uri
